Rivals Camera Stabilizer

This program takes Rivals of Aether footage and stabilizes the camera. This can make it easier for editors to automate their own camera work. It may also serve as a preprocessing step for other computer vision or AI work.

Input file format is flexible; output/tracker file formats must be MP4 or AVI. You may optionally mess with the advanced parameters.

Resolutions: for the most part, we recommend using the default settings. However, you may need to increase the amount of padding depending on how much camera movement there is. If the program is running too slowly for your purposes, you can consider lowering the processing resolution. The output resolution determines the final (potentially rescaled) resolution of the output file.
